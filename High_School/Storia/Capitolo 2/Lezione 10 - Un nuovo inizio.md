
Inizio ==Età Arcaica== (VIII-VII secolo a.C.) -> Rinascita scrittura in Grecia (==alfabetica==, suoni = parole) -> Eredità dai Fenici (relazioni commerciali) -> Ampia produzione opere letterarie (comprensione periodo) -> Aristocratici detengono terre coltivabili -> Necessità di difendere queste terre -> Nascita nuovo soldato: l'==oplita== (contadino) -> Oplita acquista propria armatura -> FALANGE oplitica (schieramento difesa, scudo protegge lato sinistro) -> Oplita forma nuovo ceto medio (liquidità per equipaggiamento) -> Nascita della polis.

==Polis== (raggruppamento comunità: valori politici, religiosi, militari) -> Spazio diviso: agropoli (parte alta, tempio divinità protettrice) <-> agorà (piazza, mercato, discussioni sociali/politiche) -> ==Cittadino Greco== (uomo libero, possesso terra o discendenza) -> Donne (formalmente cittadine, no diritti politici/cariche) -> Schiavi (non considerati persone) -> Cittadini = soldati -> Altre comunità diverse: ethné (tribù, spirito militare, santuari) -> koina (stati federali, doppia cittadinanza, partecipazione guerre).

Diverse forme di governo: aristocrazia -> democrazia -> monarchia -> Aristocratici: non solo ricchi, ma aretè (onore, rispetto, capacità battaglia) -> ==Aretè==: potenzialmente per chiunque, ma spesso ereditaria -> Mentalità agonale e competitiva (solo i migliori = rispetto/eroi -> Olimpiadi) -> Nuova forma di governo: ==tirannia== -> Tiranni: aristocratici a favore contadini-soldati -> Sviluppo commercio estero -> Programmi lavori pubblici per i più poveri -> Esempio tiranno: Pittaco di Mitilene (riforma leggi, risolve conflitti Atene, sette sapienti).
